# FastApi_ChatBot
用Fastapi搭建的一个简单的AI应用Demo

# Quick Start
## 填写自己的APIKey
## 启动FastApi
```
uvicorn Fastapi_chatbot:app --port 8080 --reload  
```
访问 docs 路由，测试接口/chat的功能
```
http://127.0.0.1:8080/docs
```
